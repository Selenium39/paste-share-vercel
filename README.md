# 文本分享应用

这是一个使用 Next.js、TypeScript、Tailwind CSS 以及 Vercel KV 构建的简单文本分享应用。

## 功能

- 创建新的文本片段  
- 生成唯一的 URL 用于分享
- 设置文本片段的有效期
- Vercel KV 后端储存数据

## 使用技术

- Next.js - React 框架
- TypeScript - 类型检查
- Tailwind CSS - UI 组件库
- Vercel KV - 后端数据库

## 说明

此仓库中的所有代码完全由 ChatGPT 4.0 生成。
